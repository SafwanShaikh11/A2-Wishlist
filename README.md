# Wishlist Management Application

A front-end web application demonstrating client-side data management with Local Storage and automated unit testing using Jest.

## Highlights

- Dynamic wishlist management
- Persistent browser storage
- Duplicate item prevention
- Automated unit testing
- Clean and responsive interface

## Built With

- HTML5
- CSS3
- JavaScript
- Jest
  
Automated unit tests have been implemented using Jest to validate the application's core functionality.

Run all tests using:

```bash
npm test
